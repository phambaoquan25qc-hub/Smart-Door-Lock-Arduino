# Smart Door Lock 🔐

A simple Arduino Smart Door Lock built with Arduino Uno.

## Features

- Password protection using Keypad 4x4
- LCD1602 display
- Servo motor door lock
- Buzzer notification
- EEPROM password storage
- Password change function

## Components

- Arduino Uno R3
- Keypad 4x4
- LCD1602
- SG90 Servo Motor
- Buzzer
- LEDs
- Breadboard
- Jumper Wires

## Default Password

1234

## Usage

### Unlock Door

1. Enter the password
2. Press #

Example:

1234#

### Change Password

1. Press A
2. Enter new 4-digit password
3. Press #

Example:

A
8888#
